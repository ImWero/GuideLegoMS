Программирование конструкатора LEGO MINDSTORMS

Начало кода

Начинать код нужно с блока который будет запускать код. Их есть 5: Первое - код запускается сразу после его запуска на роботе, Второе - когда датчик цвета видит определный цвет, Третий - когда кнопка нажата, Четвертый - когда угол менее, больше или равно определенному углу, Пятый - когда цвентральная кнопка на блоке EV3 нажата.

    Разделы:

  • Событие
  • Управление

Управление моторами

Моторами можно управлять по разному. Можно перемещать по часовой и против часовой на n-количество оборотов или секунд, можно его просто запустить без точного количества действий, можно остановить, устновить скорость в % для моторов, и можно задавать количество оборотов со определенной скоростью

    Разделы:

  • Моторы
  • Движение

Управлние датчиками

С помощью датчиков мы можем делать интересные программы. Управлять датчиками мы можем с помощью двух разделов в программе EV3 Classroom это разделы датчики и операторы. Для чего нужен раздел датчики понятно из названия, а раздел операторы нужны для задования условий. Например ехать назад если яроксть меньше 20%(для этого используется датчик цвета).

    Разделы:

  • Датчики
  • Операторы

Визуальные эффекты

Если визуальные эффекты как - настройка дисплея и звука. В EV3 Classroom есть разделы - Дисплей и Звук. Благодоря ним мы можем отображать лицо на Программируемом блоке EV3 а также чтобы он издавал звуки при необходимых условиях.

    Разделы:

  • Дисплей
  • Звук
